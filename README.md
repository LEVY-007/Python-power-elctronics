# Python for Power Electronics

Analysis of DC-DC Power Converters using Python.

## Scripts
- `converter_analysis.py` — Plots output voltage vs duty cycle for Buck, Boost and Buck-Boost converters. Includes numerical analysis at key duty cycle points.

## Converters Covered
- Buck Converter: Vout = D × Vin
- Boost Converter: Vout = Vin / (1-D)  
- Buck-Boost Converter: Vout = -(D/(1-D)) × Vin

## Tools
Python, NumPy, Matplotlib
